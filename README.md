## Features:
* Join room: Joins the room entered on the text box above.
* Disconnect: Leaves the room that you are in.
* Quit Game: Closes the game.
* Third person/ FPC: Toggles between 3rd person and FPC, can also be toggled by clicking the "Q" key
* Gamemode Selecter. Changes to selected gamemode. Opens by clicking " ` '
