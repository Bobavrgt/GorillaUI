# GorillaUI
![image](https://github.com/user-attachments/assets/94715141-7842-42e5-91d7-39915d59284c)


## Features:
* Join room: Joins the room entered on the text box above.
* Disconnect: Leaves the room that you are in.
* Quit Game: Closes the game.
* Set Modded: Sets the gamemode to modded casual.
* Set Casual: Sets the gamemode the casual.
* Third person/ FPC: Toggles between 3rd person and FPC.

  ### Why does this exist?
- Personal use for testing mods.

  ### Then why is it on your github?
- Incase someone wants it.
 
  ### How do I use it?
- Use a freecam mod, dont play with it just use it to test mods.
